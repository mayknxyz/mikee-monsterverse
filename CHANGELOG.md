## [0.1.0] - 2025-11-28

### Added

- Pure semantic HTML/CSS website (no JavaScript, no frameworks)
- Homepage with hero section and monster statistics
- 12 AI-generated monsters: Nyxos, Volcorg, Arboros, Cryos, Kukulkan, Abyssal Maw, Azmar, Fartasma, Pip, Ryukomi, Umbra'Xen, Echidnox
- 1 classic monster: King Kong
- Category pages for AI-Generated, Classics, and My Collection
- Monster detail pages with 2-column sticky image layout
- Custom 404 error page
- Responsive design for mobile, tablet, and desktop
- Dark mode styling with CSS custom properties
- Mobile navigation using HTML `popover` attribute (no JavaScript)
- Print stylesheet for paper-friendly output
- Skip-to-content accessibility link for keyboard navigation
- SEO files: `robots.txt` and `sitemap.xml`
- Open Graph and Twitter Card meta tags for social sharing
- MIT license for website code (content remains copyrighted)
- Project documentation: `README.md` with story and setup instructions
- Content tracking: `TRACKER.md` for monster inventory
